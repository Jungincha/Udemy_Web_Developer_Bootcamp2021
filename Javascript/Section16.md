# Section 16 Javascript Decision Making

#### Truthy and Falsy values

- Falsy values:
  - false
  - 0
  - ""(empty string)
  - null
  - undefined
  - NaN
- Everything else is truthy

#### How to check Truthy and Falsy
```js
if (value) {
  // When the value is Truthy
  console.log('True')
} else {
  // When the value is Falsy
  console.log('False')
}
```
